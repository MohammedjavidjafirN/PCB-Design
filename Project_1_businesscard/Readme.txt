Project description:
This project is a 555 timer-based LED blinker circuit implemented on a compact PCB designed in the form of a business card. The circuit operates using a timer IC configured in astable mode, where it continuously generates a square wave signal without requiring any external trigger.
When powered by the onboard coin cell battery, the timer IC alternates between charging and discharging cycles through a resistor-capacitor (RC) network. This periodic charging and discharging action produces a continuous oscillating output at the timer’s output pin. As a result, the connected LED turns ON and OFF repeatedly, creating a visible blinking effect. The blinking rate is determined by the values of the resistor and capacitor used in the timing circuit. A slide switch is incorporated to control the power supply, allowing the user to turn the circuit ON or OFF as needed.
